#9th_1team_FE

밸런스게임 프론트 프로젝트

## How to use

작업전에 .env 세팅 필수

```bash
cp .env.sample .env
```

스키마 다운로드 받아야됨

```bash
npm run schema
# or
yarn schema
```

다운로드 받은 스키마 기준으로 relay 제너레이트 작업 해야됨

```bash
npm run relay
# or
yarn relay
```

프로젝트 시작

```bash
npm run dev
# or
yarn dev
```
